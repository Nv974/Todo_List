## Installation

Pré-requis : il faut avoir NodeJs installé sur sa machine https://nodejs.org/en/

1. Installer les packages </br>
   Lancer la commande suivante dans votre terminal : </br>
   npm install ou yarn
   
3. Lancer le serveur </br>
   npm start ou yarn start
